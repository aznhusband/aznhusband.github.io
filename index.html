Current Version: 1.9.4

Repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://aznhusband.github.io/repo Give it the name "aznhusband-repo".
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "aznhusband-repo", and install repository.aznhusband-kodi-repo-1.0.1.zip
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "aznhusband-kodi-repo"
  • Go into the "Video add-ons" section in the repo, and you'll find the Icdrama plugin.

Direct download

If you'd rather not add the repo, you can directly install the addon by adding the following location in the Kodi file manager:

https://aznhusband.github.io/plugin.video.icdrama/

UrlResolver

This plugin depends on UrlResolver version 3.0.34 or better. It should install automatically if you add the aznhusband repo.

If you'd rather install it manually, you can find a copy that works with Icdrama at: https://aznhusband.github.io/script.module.urlresolver. You can add this location as a source in the file manager, and install the zip from there.
